# Xin chào! 👋

Đây là bài viết đầu tiên trên blog của tôi. Tôi rất vui khi được chia sẻ những suy nghĩ và kinh nghiệm của mình với mọi người.

## Tôi viết gì ở đây?

Blog này sẽ tập trung vào các chủ đề:

- **Lập trình web** — React, TypeScript, và các công nghệ hiện đại
- **Kiến trúc phần mềm** — Cách tổ chức code sao cho dễ bảo trì
- **Công cụ & workflow** — Những thứ giúp tôi làm việc hiệu quả hơn

## Một đoạn code ví dụ

Đây là một function đơn giản trong TypeScript:

```typescript
function xinChao(ten: string): string {
  return `Xin chào, ${ten}! Chào mừng đến với blog của tôi.`;
}

console.log(xinChao("bạn"));
// → Xin chào, bạn! Chào mừng đến với blog của tôi.
```

</Callout>

## Toán học với KaTeX

Blog này hỗ trợ công thức toán học. Ví dụ, công thức Euler nổi tiếng:

$$
e^{i\pi} + 1 = 0
$$
